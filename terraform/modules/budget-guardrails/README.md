# budget-guardrails

Creates AWS Budgets-based cost guardrails for SAMSON.

## Default Controls

- Monthly budget limit: $120
- Actual spend alert: 100% ($120)
- Forecast alert: 125% (~$150)
- Emergency forecast alert: 146% (~$175)

## Purpose

Provides proactive cost monitoring and alerting to ensure the platform remains within a controlled operating budget.

## Enterprise Relevance

Cost governance is a foundational cloud control. This module demonstrates that security architecture is designed with financial accountability and operational discipline in mind.
