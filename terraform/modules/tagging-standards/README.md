# tagging-standards

Defines mandatory tags used across SAMSON to support governance, cost allocation, ownership, and compliance.

## Required Tags

- Project
- Environment
- Owner
- CostCenter
- DataClassification
- ComplianceScope
- ManagedBy
- Repository

## Purpose

Standardized tagging improves:
- Cost visibility
- Ownership accountability
- Compliance reporting
- Automation and inventory management

## Enterprise Relevance

Consistent metadata is a foundational governance control used to enforce operational standards and support audits.
